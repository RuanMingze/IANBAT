# IANBAT

超难人机验证系统 - 一个极其困难的人类验证挑战平台

## 简介

IANBAT 是一个专为筛选真正人类设计的超难人机验证系统。通过 20 个精心设计的关卡，挑战你的反应速度、记忆力、精准度和认知能力。

**警告：本系统难度极高，不适合用于正常网站的人机验证。**

## 特性

- 20 个独特的验证关卡
- 支持 CDN 嵌入模式
- 排行榜系统
- 深色主题设计
- 支持 postMessage 跨域通信

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Supabase

## 开发

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
```
